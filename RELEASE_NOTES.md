## ClashFX 1.0.11

### Improvements

- **Redesigned tray icon picker UI** — Clean horizontal layout with icon preview well, description text with recommended image size, and always-visible "Reset to Default" button (greyed out when no custom icon is set).
- **Fixed "Tray Icon" terminology** — Changed to "Menu Bar Icon" in Simplified Chinese to match macOS conventions.
- **Added image size guidance** — The icon picker now shows recommended dimensions (36×36 px / 72×72 for Retina @2x, PNG format).

---

### 改进

- **重新设计菜单栏图标选择器 UI** — 采用水平布局，包含图标预览、带建议尺寸的说明文字，以及始终可见的「恢复默认」按钮（无自定义图标时置灰）。
- **修正术语** — 简体中文「托盘图标」改为更符合 macOS 习惯的「菜单栏图标」。
- **新增图片规格提示** — 图标选择器显示建议尺寸（36×36 像素 / Retina @2x 为 72×72，PNG 格式）。
